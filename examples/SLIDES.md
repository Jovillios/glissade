# glissade

Convert Markdown to Interactive Slide Presentations

*A simple, self-contained HTML slide generator*

---

# What is glissade?

**glissade** is a Python tool that transforms markdown files into beautiful, interactive HTML presentation slides.

- **Markdown-based** : Write slides in familiar markdown syntax
- **Standalone** : Single self-contained HTML file
- **Customizable** : Multiple Catppuccin color themes
- **Interactive** : Keyboard navigation, live reload mode
- **Fast** : No external dependencies or build tools

> Perfect for quick presentations, documentation, or sharing ideas.

---

# Getting Started

**Three simple steps:**

1. Install: `pip install glissade`
2. Create: `glissade --input SLIDES.md --output index.html`
3. Open: Launch `index.html` in your browser

Ready in seconds. No setup required.

---

# Use Cases

- **Technical talks** : Present code, slides, and demos

# Try It Now

1. Edit `SLIDES.md` in your editor
2. Run `glissade --live`
3. Open `http://localhost:8080` in your browser
4. Edit and watch it update live

That's it! No build steps, no compilation, no fuss.
